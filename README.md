 Dotfiles
==========
These are some of my dotfiles, I primarily use this to back up my own stuff, but feel free to have a look and/or use whatever I have here. :P

 Installation
--------------
Run:

	git submodule init
	git submodule update
	./install.sh

 Dependencies
--------------

 - Tmux
 - Python 3
 - Oh-my-zsh
